## 追加検討中の要素

科学大ポータル(東工大ポータルではない)の2段階認証における自動ログインの有効化

## 追加しない予定の要素

ユーザー名、パスワードの自動入力(セキュリティー上の観点よりパスワードマネージャーを使用することをお勧めします)
